Google Fonts

Google Fonts (formerly known as Google Web Fonts) is an interactive directory of more than a thousand free and open source font families, an interactive web directory for browsing the library, and APIs for using the fonts via CSS and Android. The website was launched in 2010 and is available in over 135 languages. Most of the fonts are released under the SIL Open Font License 1.1, while some are released under the Apache License, both of which are free licenses. The fonts are also served by Monotype’s SkyFonts, and Adobe’s Edge Web Fonts and Typekit services
