Tags
HTML tags are the keywords on a web page that define how your web browser must format and display your web page. Almost all tags contain two parts, an opening, and a closing tag.
Eg.
  • Bold
  • Ittalic
Elements
HTML element is a component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document
Eg.
  • Paragraph content
  • heading content.
  • division content.
  • break content.
Attributes
HTML attribute is a piece of markup language used to adjust the behavior or display of an HTML element.
Eg.
  • hrf attribute
  • src attribute
